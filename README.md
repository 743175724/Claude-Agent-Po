# My Agents Project

A curated collection of Claude agent definitions, commands, and skills optimized for a Windows-centric yet multi-disciplinary development workflow (drivers, UI/UX, reverse engineering, web, and game development).

## Structure
```
my-agents-project/
├── .claude-plugin/
│   └── marketplace.json
├── plugins/
│   ├── project-management/
│   │   ├── agents/
│   │   ├── commands/
│   │   └── skills/
│   └── quality-assurance/
│       ├── agents/
│       ├── commands/
│       └── skills/
├── docs/
│   └── usage.md
└── README.md
```

## Highlighted Roles
- **Intelligent Task Dispatcher:** Routes work, plans milestones, and keeps multi-platform efforts synchronized.
- **Bug Fix Specialist:** Executes deep diagnostics and remediation across drivers, UI frameworks, games, and web apps.
- **Senior Code Reviewer:** Applies rigorous review standards with an enforceable rework loop prior to approval.

## Getting Started
1. Browse `.claude-plugin/marketplace.json` to discover available plugins.
2. Follow `docs/usage.md` for a recommended workflow integrating dispatch, bug fixing, and review.
3. Extend by adding new plugins tailored to engines, toolchains, or platforms you work with frequently.
