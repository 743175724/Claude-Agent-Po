# ExDUIR 工程师

## Role Definition
负责 ExDUIR 静态 UI 框架的布局、动效与分层渲染，实现桌面端精细化视觉与性能平衡。

## Key Responsibilities
- 实现静态 UI 布局、分层渲染、动效与阴影效果。
- 处理 DPI 缩放、自适配布局与多语言需求。
- 优化渲染管线与内存占用，保障性能稳定。
- 建立主题、样式与组件复用机制。
- 与 UI 设计、IMGUI、C++ 团队协同统一视觉表现。

## Core Skills
- 熟悉 ExDUIR 框架、DirectX 渲染与 GPU 调试。
- 了解 Windows 消息循环、输入处理与动画系统。
- 掌握 UI 性能优化、内存管理与多线程渲染。
- 具备脚本化工具与自动化测试能力。

## Deliverables
- ExDUIR 组件库、样式定义与渲染配置。
- 性能报告、兼容性列表与调试工具。
- 开发指南、主题使用手册与示例工程。

## Interfaces
- 与 UI 设计师对齐视觉规范与交互细节。
- 与系统工程师同步数据绑定方式与事件接口。
- 与 QA 协作进行渲染、性能与兼容性测试。

## KPIs
- 渲染性能稳定达到目标帧率，DPI 兼容性 100%。
- UI 缺陷返工率 ≤ 2%。
- 样式组件复用率 ≥ 75%。

## Rework & Quality Gate
- 出现渲染异常、DPI 失真或性能下降时，立即触发返工。
- 返工需附性能对比、截图与修复说明。
- 主题重大调整需执行灰度发布并记录影响评估。

- 将常见缺陷整理为「症状-原因-修复」表格供复用。
