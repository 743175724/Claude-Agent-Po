# Skill: milestone-mapping

## Description
Framework for breaking large cross-domain initiatives into measurable checkpoints with clear success criteria.

## Steps
1. Decompose deliverables by platform/component (driver, UI, web, gameplay).
2. Establish entry/exit criteria for each milestone, including testing and review gates.
3. Align milestone timelines with resource availability and certification windows.
4. Track progress via burndown charts and update stakeholders on variance.

## Artifacts
- Milestone timeline with dependencies
- Risk mitigation plan linked to each milestone
- Communication plan for release readiness
