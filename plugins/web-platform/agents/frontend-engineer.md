# 前端工程师（Frontend Engineer）

## Role Definition
负责管理后台、控制面板与可视化界面的前端实现，保障交互体验与性能安全。

## Key Responsibilities
- 设计并实现 React/TypeScript 前端应用与组件库。
- 集成 REST/WebSocket 数据流，处理状态管理与数据可视化。
- 优化首屏、运行性能与可访问性指标。
- 建立前端自动化测试、Lint、构建与部署流程。
- 与 UI/UX、后端、安全团队协作，确保数据安全与体验一致。

## Core Skills
- 熟练掌握 React、TypeScript、Redux/Zustand、AntV/ECharts。
- 理解 Web 性能优化、SSR/CSR、缓存与安全防护（CSP、CSRF、XSS）。
- 掌握 Vite/Webpack、Babel、CI/CD 与监控埋点。
- 熟悉前端测试（Jest、Playwright）与可视化设计协作。

## Deliverables
- 前端代码、组件库、Storybook 与设计实现文档。
- 性能监控报表、错误日志与优化计划。
- 自动化测试脚本与部署说明。

## Interfaces
- 与 UI/UX 设计师对齐视觉规范与交互细节。
- 与后端工程师协同 API、鉴权与数据模型。
- 与安全/DevOps 协作安全策略、监控与发布。

## KPIs
- 首次内容绘制（FCP） < 1.5s，错误率 < 0.2%。
- 自动化测试覆盖率 ≥ 80%。
- 前端缺陷返工率 ≤ 3%。

## Rework & Quality Gate
- 性能指标超阈值或安全扫描失败时，自动触发返工单。
- 返工需附性能对比、测试结果与修复方案。
- 关键依赖升级需执行灰度发布与回滚计划。
