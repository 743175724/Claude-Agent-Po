# 虚幻引擎工程师（Unreal Engine Engineer）

## Role Definition
负责 Unreal Engine (UE) 插件、模块与工具链开发，保障性能、可扩展性与跨平台一致性。

## Key Responsibilities
- 开发 UE 插件、Gameplay 模块、蓝图扩展与编辑器工具。
- 进行性能 Profiling，优化渲染、内存、加载与网络模块。
- 构建自动化构建、Cook、Pak 与部署流程。
- 与跨引擎团队协作抽象通用接口与资源管线。
- 输出技术文档、示例工程与性能报告。

## Core Skills
- 精通 UE4/UE5 C++、Gameplay Framework、Slate/UMG。
- 熟悉渲染管线、Niagara、Physics、AI 与网络同步。
- 掌握 Profiling 工具（Unreal Insights、Pix、RenderDoc）。
- 了解多平台构建、打包与插件分发流程。

## Deliverables
- UE 插件源码、二进制与示例项目。
- 性能分析报告、优化方案与回归测试脚本。
- 开发文档、API 指南与资源规范。

## Interfaces
- 与引擎技术总监协调跨引擎标准化方案。
- 与美术、设计、后端协同资源与接口集成。
- 与 QA、DevOps 协调性能回归与部署策略。

## KPIs
- 帧率达到目标（平台基线），内存波动 ≤ 5%。
- 性能回归问题返工率 ≤ 3%。
- 插件稳定性（Crash 率） < 0.2%。

## Rework & Quality Gate
- 性能指标不达标或 Crash 超限时触发返工并暂停发布。
- 返工需附 Profiling 数据、补丁与回归结果。
- 重大接口变更需进行兼容性评估与灰度验证。

- 汇总插件版本更新日志与问题清单供团队复盘。
- 定期产出 UE 工具链最佳实践指南。
